# **App Name**: STAYLY

## Core Features:

- User Authentication with Firebase: Secure user sign-up and sign-in using Firebase Authentication. Store additional user data in MongoDB.
- Property Listings: Allow users to list rental properties with details such as title, type, location, price, and amenities. Save listings to MongoDB.
- Roommate Listings: Enable users to list shared accommodations with specific details. Store roommate listings in a separate MongoDB collection.
- Property Search and Filtering: Implement a search bar to find properties based on location, property type, or keywords. Include filters for short-term, long-term, and roommate listings.
- Dynamic Property Cards: Display property information in cards with image, title, location, price, and basic amenities. Fetch data dynamically from MongoDB.
- Property Details Page: Show detailed information about a selected property, including owner/renter contact details, description, amenities, photos, and reviews.
- Theme Switcher: Allow users to switch between black-gold dark mode and white-gold light mode, storing the preference in localStorage.

## Style Guidelines:

- Primary color: Gold (#D4AF37) for luxury and exclusivity.
- Background color: Near-black (#121212) for dark mode, desaturated gold (#FAFAFA) for light mode.
- Accent color: Darker gold (#B8860B) for hover effects and subtle highlights, offering a high-end touch without overwhelming the user.
- Body and headline font: 'Poppins', a geometric sans-serif for a modern, fashionable aesthetic.
- Use elegant icons with gold accents to represent amenities and property features.
- Use a grid layout for property listings to ensure a clean and organized display.
- Incorporate smooth transitions and subtle animations for loading states and UI interactions.
