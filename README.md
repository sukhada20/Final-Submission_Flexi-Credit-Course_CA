# üè° STAYLY ‚Äì Rental Property Listing Website

**STAYLY** is a full-stack rental property platform that lets users **browse, list, and rent** properties for short-term or long-term stays and also **find roommates**.  
It is built with **React.js**, **Node.js**, **Express.js**, **MongoDB Atlas**, and **Firebase Authentication**, featuring a sleek **Black-Gold theme** with a switchable **White-Gold light mode**.

## ‚ú® Features

### üîπ User Experience
- Beautiful **animated splash screen** showing the STAYLY logo before entering the site.
- **Firebase Authentication** for secure Sign In / Sign Up with Email & Password.
- Smooth user flow ‚Äî access to the home page only after authentication.
- Toggle between **Dark (Black-Gold)** and **Light (White-Gold)** themes.

### üîπ Core Functionality
- **Browse Properties:** View all listed properties with filters for short-term, long-term, and roommate options.
- **List Property:** Post new rental properties with complete details, including photos and pricing.
- **Find Roommate:** Post or search listings for shared accommodations.
- **Search Bar:** Smart search for location, property type, or keywords.
- **Filter Tabs:** Quickly switch views ‚Äî All Properties | Short-Term | Long-Term | Roommate.
- **Property Details Page:** Displays full information, owner contact, amenities, and reviews.
- **Footer Section:** Quick links for renters and landlords.

### üîπ Backend and Database
- **Node.js + Express.js** backend API.
- **MongoDB Atlas** for storing users, properties, and roommate listings.
- Secure route protection with **Firebase Auth tokens**.
- Dynamic property fetching and posting using REST APIs.
